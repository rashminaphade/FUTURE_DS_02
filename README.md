# FUTURE_DS_02
Customer Churn Analysis Dashboard built using Power BI to analyze churn patterns and customer retention insights. The dashboard highlights key KPIs such as total customers, churn rate, average tenure, and provides interactive visualizations based on contract type, gender, tenure range, and payment methods to support data-driven decision making.
